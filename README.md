# Word Memorizer for SillyTavern
在等待 AI 回复时弹出单词记忆面板，支持**自定义词库导入**，自动识别单词、词性、释义。手机端完美适配。

## 功能亮点
- 📘 独立浮动面板，不影响聊天，可拖动
- 📥 一键导入自定义词库：粘贴 `1.good adj.好的` 等格式自动解析
- 🔄 测试模式：随机抽词，点击翻转查看释义，标记掌握/未掌握
- 💾 学习进度和词库自动保存到浏览器本地
- 📱 手机端按钮、字体加大，触摸友好

## 安装方法
1. 进入酒馆的 **插件管理 (Extensions)** 页面
2. 点击 **安装拓展 (Install extension)**
3. 输入本仓库地址：`https://github.com/yliu63448-hub/sillytavern-word2.0`
4. 安装范围选择 **Install for all users**
5. 刷新页面

- 点击聊天界面右上角的 📘 图标打开面板
